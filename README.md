# hono-fs-router

A file based API Router for Hono.

## Local Development

Uses [Bun](https://bun.sh) v1.2.5.

|     Command     | Usage                                 |
| :-------------: | :------------------------------------ |
|  `bun install`  | Install dependencies                  |
| `bun run build` | Build the core library                |
|  `bun run dev`  | Use for run sample server for testing |
